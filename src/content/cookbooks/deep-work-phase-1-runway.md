**Role: The Cognitive Performance Coach.** Objective: eliminate the friction between sitting down and actually starting — so the first minute of your session is execution, not orientation.

## The Recipe

```
Act as a cognitive performance coach and behavioral psychologist specializing in flow-state optimization. I am about to enter a high-leverage Deep Work session and need to eliminate cognitive friction, clear my working memory, and map an ironclad execution path.

My core task/deliverable for this session is: [INSERT TASK, e.g., Refactoring an eBPF network logging script, sketching 9 garden bed layouts]
My available time block is: [e.g., 90 minutes, 2 hours]
My biggest current focus vulnerability right now is: [e.g., Checking email tabs, phone notifications, mental fatigue]

Please build a customized "Pre-Session Runway" for me:
1. The Single Definition of Done: Reframe my messy task into an objective, binary success threshold for this block (e.g., "By the end of this session, X script compiles without error logs" instead of "Work on script").
2. The Environmental Sanity Check: Give me a 3-step physical and digital environmental sweep tailored to my explicit focus vulnerability to prevent mindless distraction loops.
3. Micro-Scaffolding: Break my session down into 3 distinct, chronologically ordered micro-milestones to build immediate dopamine momentum within the first 15 minutes.

Deliver my runway so I can lock in my environment and begin.
```

## The three runway components

| Component | What it eliminates |
|---|---|
| Single Definition of Done | Scope ambiguity — "work on X" is not a completable task; a binary threshold is |
| Environmental Sanity Check | Your specific distraction vector — tailored to what you named, not generic advice |
| Micro-Scaffolding | The blank-page paralysis of the first 15 minutes — momentum is built in small steps |

## Why "Definition of Done" is the most important output

"Work on the script" can never be finished, so your brain never fully commits to it. "By the end of this session, the authentication module compiles without errors and passes the two existing test cases" is completable — which means you can actually enter a flow state around it. Vague tasks generate vague attention.

## The focus vulnerability input

This is the input that makes the Environmental Sanity Check non-generic. "Mental fatigue" produces different environmental guidance than "checking email tabs" or "phone notifications" or "an unresolved conversation I'm replaying." Name the actual vulnerability you're walking in with.

## The 15-minute momentum window

The first 15 minutes of a deep work session are the highest-friction period — your brain is still resisting the switch from ambient to focused mode. Micro-milestones that are deliberately easy ("open the file and read the last 10 lines," "write one sentence") build the forward momentum that carries you into the harder middle section without requiring a separate act of will.
