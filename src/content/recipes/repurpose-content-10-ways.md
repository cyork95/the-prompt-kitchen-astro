**One great idea, distributed poorly, reaches nobody. One great idea, atomized properly, reaches everyone.** This recipe extracts 10 distinct, platform-native assets from a single piece of content — each formatted for where it lands, not copy-pasted from the original.

## The Recipe

```
Act as an advanced multimedia distribution strategist and content operations architect. I want to build a high-leverage repurposing blueprint to atomize a single long-form piece of content into 10 distinct, platform-native micro-assets.

Please ask me to paste my long-form content pillar (e.g., a newsletter essay, blog post, or video transcript). Once I provide it, extract the insights and generate:
1. An Educational X/Twitter Thread (A 5-tweet tactical breakdown).
2. A Short-Form Video Script (A 45-second fast-paced script for Reels/TikTok).
3. A High-Authority LinkedIn Post (Leading with a personal observation or framework).
4. A Deep-Dive Newsletter Teaser (An intrigue-driven curiosity email).
5. An Infographic Blueprint (A textual text-to-visual layout plan for a designer).
6. A Q&A Community Post (A thought-provoking discussion prompt for Reddit/Discord).
7. A Contrarian Hook Post (A short text post challenging industry dogma).
8. A "Before vs. After" Case Study Blurb.
9. A Key Takeaway Checklist (Optimized for quick saves).
10. A Micro-Audio Script (A 60-second narrative audio voiceover).

Give me the step-by-step instructions on how we will process my text.
```

## The 10 assets and their platforms

| Asset | Platform | Format rule |
|---|---|---|
| X Thread | Twitter / X | 5 tweets, one idea each, hook in Tweet 1 |
| Short-Form Video Script | Reels / TikTok | 45 seconds, hook in first 2 seconds |
| LinkedIn Post | LinkedIn | Opens with observation, no fluff opener |
| Newsletter Teaser | Email | Curiosity-driven, earns the click |
| Infographic Blueprint | Design tools | Layout plan for a designer — text-to-visual |
| Community Post | Reddit / Discord | Discussion prompt, not a broadcast |
| Contrarian Hook | Any text platform | Challenges a status quo assumption |
| Case Study Blurb | Any | Before → After in 3–4 sentences |
| Checklist | Save-optimized posts | Bullet list of takeaways |
| Micro-Audio Script | Podcast clip / voiceover | 60-second narrative arc |

## Why "platform-native" matters

Copying the same text to every platform is how content gets ignored. Each platform has unwritten formatting rules — LinkedIn rewards personal observation openers; TikTok rewards immediate hooks; Reddit rewards genuine discussion questions. The recipe enforces these norms for each asset.
